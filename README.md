# Object-oriented Programming 1

This repository contains materials and tasks related to the "Object-oriented Programming 1" course offered in the fourth semester of Applied Computer Science at AGH University of Krakow.

## Course Topics

- Conversions and Operators (Extended Usage)
- Inheritance (Extended Usage)
- Function and Class Templates
- Standard Template Library (STL)
- Extending C++ Language
- Static Polymorphism and Meta-programming using Templates
